# Chisa – The Boundless Analyst

A character showcase page for Chisa from Wuthering Waves, featuring an immersive autumn-themed design with animations and interactive elements.

## Features

- Responsive design with Tailwind CSS
- Animated background video with autumn temple theme
- Particle effects (falling leaves)
- Smooth scroll animations
- Interactive gallery with modal
- Character information and resonance bonds sections

## Live View

View the live site at: https://zaichi234509.github.io/chisa/

## Usage

Open `chisa.html` in a web browser to view the page locally.
